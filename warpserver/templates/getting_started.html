Getting Started

To Get Started With playing Dockingstation online.
First you, have to install Dockingstation, It should work with all Versions including the standalone version you can get at here.
Second, you need an Account on AlbianWarp, get yourself one right here.
Finally to be able to go online you must download the server.cfg and place it in the main directory of DockingStation.
The server.cfg should contain the following: 


"Server 0 FriendlyName" {{ server_name }}
"Server 0 Host" {{ host }}
"Server 0 ID" 1
"Server 0 Port" {{ port }}
"Server Count" 1
With the Configuration in place and an Account created, you should now be able to go online like you used to, back in the Days!

What to do, if you used a previous Version of AlbianWarp ?

If you used a previous version of AlbianWarp you must first remove the Game Modifications, located in the Bootsrap folder.

  1. Remove the whole directory: "050 Albian Warp" located in your Docking Stations Bootstrap Folder.
  2. Remove the file "AW eam vars.cos" from the Directory "./Bootstrap/000 Switcher"
  3. Remove the file "reinject_updated_agents.cos" from the Directory "./Bootstrap/dsbuild 195" and restore the original by renaming the Baclup file "reinject_updated_agents.cos.backup" to "reinject_updated_agents.cos".
This should remove all the AlbianWarp Game Modifications, now you just need to create a New World
